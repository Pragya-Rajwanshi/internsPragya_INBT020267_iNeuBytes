# 📖 Project Overview

The Movie Recommendation System is an AI-powered web application that recommends movies based on a Hybrid Recommendation approach. The system combines recommendation techniques to provide relevant movie suggestions and enhances the user experience by using Google's Gemini AI to explain why each movie is recommended.

The application is developed using Python and Flask and utilizes the MovieLens dataset for generating personalized recommendations.



# 🎯 Objectives

- Build an intelligent movie recommendation system.
- Recommend similar movies using Hybrid Recommendation.
- Integrate Gemini AI for natural language explanations.
- Develop a responsive web interface using Flask.
- Demonstrate practical implementation of AI and Machine Learning concepts.

# 💻 Installation

Clone the repository

```
git clone <repository-url>
```

Go inside project folder

```
cd Major Project
```

Install dependencies

```
pip install -r requirements.txt
```

Create .env file

```
GEMINI_API_KEY=YOUR_API_KEY
```

Run model training

```
python train_model.py
```

Start Flask application

```
python app.py
```

Open browser

```
http://127.0.0.1:5000
```
