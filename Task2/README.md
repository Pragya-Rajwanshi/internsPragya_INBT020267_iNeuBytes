# Sentiment Analysis using Machine Learning & Deep Learning

## Project Overview

This project performs sentiment analysis on movie reviews using three different models:

- Logistic Regression
- Support Vector Machine (SVM)
- Long Short-Term Memory (LSTM)

The models classify reviews as Positive or Negative.

---

## Dataset

Dataset: IMDb Movie Reviews Dataset

- 50,000 movie reviews
- Balanced dataset
- 25,000 Positive
- 25,000 Negative

## Machine Learning Models

### Logistic Regression

- TF-IDF Features
- Accuracy ≈ 88.7%

### Support Vector Machine

- TF-IDF Features
- Accuracy ≈ 88.9%

### LSTM

- Embedding Layer
- LSTM Layer
- Dropout Layer
- Dense Layer
- Accuracy ≈ 88%
